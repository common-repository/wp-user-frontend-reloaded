== Description ==

Based on WP User Frontend by Tarek Hasan. This plugin adds more features to the original plugin such as private posts and custom taxonomies.

The following features are added.

* Users are able to set their defaut post status to 'Private'.
* Users have the choice to set their posts to custom taxonomies as well as categories. 


== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
